# 🧮 Simple Calculator – Software Testing Task

This project was completed as part of my **Software Testing Internship at Prodeology**.  
The goal of this task was to test and validate the functionality of a web-based calculator application.

---

##  Project Description

The calculator performs the following operations:

- Addition
- Subtraction
- Multiplication
- Division
- Decimal calculations
- Error handling

The main focus of this task was **manual testing and functional validation**.

---

##  Technologies Used

- HTML
- CSS
- JavaScript
- Manual Testing Techniques

---

##  Test Scenarios Covered

- Positive number calculations
- Negative result validation
- Decimal operations testing
- Operator precedence (BODMAS rule)
- Division by zero validation
- Invalid input handling

---

##  Testing Types Performed

- Functional Testing
- Boundary Testing
- Negative Testing
- UI Testing

---

##  How to Run the Project

1. Download or clone this repository.
2. Open the folder in VS Code.
3. Open `index.html`.
4. Run using Live Server or open directly in a browser.
